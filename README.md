# bysykkel-client-java

Denne løsningen er testet på Linux (Ubuntu 18.04) og Mac OS (Mojave 10.14.3) med,
```
javac 11.0.1
Apache Maven 3.6.0
```

## Run
```
$ mvn clean install
$ java -jar target/no.sasoria.losning-1.0-SNAPSHOT.jar
```
